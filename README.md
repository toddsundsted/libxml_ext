Extensions to XML in Crystal.

This library wraps [libxml2](http://xmlsoft.org/) functionality for
mutating XML documents.

## Installation

1. Add the dependency to your `shard.yml`:

```yaml
dependencies:
  libxml_ext:
    github: toddsundsted/libxml_ext
```

2. Run `shards install`

## Usage

```crystal
require "libxml_ext"
```
