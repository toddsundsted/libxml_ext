    libxml_ext

    Extensions to XML in Crystal.

    This library wraps libxml2 functionality for mutating XML documents.

    Installation

    1. Add the dependency to your shard.yml:
    dependencies:
  libxml_ext:
    github: toddsundsted/libxml_ext
    1. Run shards install

    Usage

    require "libxml_ext"
